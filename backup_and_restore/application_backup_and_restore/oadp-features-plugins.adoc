== OADP support for {iibm-cloud-bm-reg} and {ibm-z-reg}

OpenShift API for Data Protection (OADP) is platform neutral. The information that follows relates only to IBM Power and to {ibm-z-reg}.

OADP 1.1.0 was tested successfully against {product-title} 4.11 for both IBM Power and {ibm-z-reg}. The sections that follow give testing and support information for OADP 1.1.0 in terms of backup locations for these systems.
